# C--SingleFileDownload

### 1. 说明

##### 		通过HTTP协议单个文件下载到定义的文件夹的控制台程序

### 2. 调用方式

````
C--SingleFileDownload\bin\Release\DownForIE.exe [json文件路径] [后缀] [保存目录]
````

````
例：C--SingleFileDownload\bin\Release\DownForIE.exe [.\Json.txt] [.dcm] [.\Dcm]
````
